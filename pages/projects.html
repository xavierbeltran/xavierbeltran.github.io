Maze Game

Date: November 2023

Course: Computer Graphics

Description: Designed and implemented a dynamic maze game in two distinct renditions, replicating a top-view scenario where a mouse navigates toward an exit. The maze that the "mouse" is navigating through is generated as a hexadecimal representation where the walls represent the bits in each cell to allow for different versions.

    The first version uses a circle and the arrow keys to move around the maze, and simple translations in WebGL.
    The second version uses a new "mouse" and the arrow keys cause for rotations as well as translations to build on the previous version.

Tools and Tech: JavaScript, Visual Studio Code, WebGL
