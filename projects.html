Information Retrieval System

Date: December 2023

Course: Information Retrieval

Description: Collaborated with two other students to build an information retrieval system that has capabilities to query a corpus filled with close to 5000 documents. The system used various different algorithms to train the data and produce a result more similar to the query, and the user has the option to choose which algorithm to be tested.

Tools and Tech: Python, PyCharm, Anaconda
